# Puntos_extra_proyect-final
COTIZADOR AUTOMATIZADO SEGUROS TK-U
Este repositorio contiene la solucion para esta empresa de seguros, automatizando el proceso

El programa como tal funciona mediante un ciclo que permite realizar cotizaciones.
El usuario debe ingresar sus datos personales y económicos y el sistema validara lo siguiente:
  * Que el asegurado sea mayo de eddad
  * Los recargos correspondientes a su núcleo familizar (se incluyen los hijos sin importar su edad)
  * Los recargos económicos por bienes (propiedades) e ingresos mensuales.

Para finalizar el programa, el usuario debe escribir la palabra "salir" al momento de ingresar su nombre.

OPCIONES DE MEJORA
Se sugiere que se cree una Interfaz Web, de los promt() a un formulario de HTML5 con validaciones en tiempo real.
Tambien se recomienda tener mayor seguridad de datos, implementando un sistema que no permita ingresar valores negativos o texto en los campos de dinero y edad.
Por ultimo se sugiere generar reportes como archivos PDF con el resumen de la cotización para el cliente, claro hace falta solicitar su NIT para terminar este ultimo paso en la cotización.
